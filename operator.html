Operators in PHP

Introduction to Operators

In PHP, operators are special symbols that perform operations on variables and values. They allow you to manipulate data, perform calculations, compare values, and much more. PHP supports several types of operators, categorized as follows:

1. Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations on numeric values. The common arithmetic operators are:

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder after division)
  • **: Exponentiation (raising a number to the power of another)

** Example:

$a = 10;
$b = 20;
$sum = $a + $b; // 30
$diff = $b - $a; // 10
$prod = $a * $b; // 200
$div = $b / $a; // 2
$mod = $b % $a; // 0
$exp = $a ** 2; // 100

2. Assignment Operators

Assignment operators are used to assign values to variables. In addition to simple assignment, they can also perform arithmetic operations and assign the result to the variable:

  • =: Simple assignment
  • +=: Add and assign
  • -=: Subtract and assign
  • *=: Multiply and assign
  • /=: Divide and assign
  • %=: Modulus and assign

** Example:

$x = 5;
$x += 10; // x = x + 10; (15)
$x -= 3; // x = x - 3; (12)
$x *= 2; // x = x * 2; (24)
$x /= 4; // x = x / 4; (6)
$x %= 2; // x = x % 2; (0)

3. Comparison Operators

Comparison operators are used to compare two values. They return a boolean value (true or false) based on the comparison:

  • ==: Equal
  • !=: Not equal
  • ===: Identical (equal in value and type)
  • !==: Not identical
  • <: Less than
  • >: Greater than
  • <=: Less than or equal to
  • >=: Greater than or equal to

** Example:

$a = 10;
$b = 20;
$result1 = ($a == $b); // false
$result2 = ($a != $b); // true
$result3 = ($a < $b); // true
$result4 = ($a >= $b); // false
$result5 = ($a === '10'); // true (same value and type)

4. Logical Operators

Logical operators are used to combine conditional statements. They return a boolean result based on the logic of the conditions:

  • &&: Logical AND (true if both operands are true)
  • ||: Logical OR (true if at least one operand is true)
  • !: Logical NOT (inverts the boolean value)

** Example:

$a = true;
$b = false;
$result1 = $a && $b; // false
$result2 = $a || $b; // true
$result3 = !$a; // false

5. Increment/Decrement Operators

Increment and decrement operators are used to increase or decrease the value of a variable by 1:

  • ++: Increment (increases value by 1)
  • --: Decrement (decreases value by 1)

These can be used as prefix or postfix:

  • ++$x: Increments $x and then returns its value
  • $x++: Returns the value of $x and then increments it

** Example:

$x = 5;
$x++; // x is now 6
$y = ++$x; // y is 7, x is 7
$x--; // x is now 6

6. String Operators

String operators are used to concatenate (join) strings together:

  • .: Concatenation operator
  • .=: Concatenation assignment operator

** Example:

$str1 = "Hello";
$str2 = "World";
$greeting = $str1 . " " . $str2; // "Hello World"
$greeting .= "!"; // "Hello World!"

7. Array Operators

Array operators are used to compare arrays. They allow you to determine the relationship between two arrays:

  • +: Union (combines two arrays)
  • ==: Equality (checks if arrays have the same key/value pairs)
  • ===: Identical (checks if arrays have the same key/value pairs in the same order and of the same type)

** Example:

$array1 = array("a" => "apple", "b" => "banana");
$array2 = array("b" => "banana", "c" => "cherry");
$result1 = $array1 + $array2; // Union
$result2 = $array1 == $array2; // Equality
