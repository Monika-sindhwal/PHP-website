Arrays in PHP

Introduction

Arrays are used to store multiple elements under a single variable. This helps reduce the effort of creating different variables for each piece of data. For example, if we want to store five names, we could create five separate variables. However, if the number of names increases to one hundred, managing those variables would be cumbersome. In such cases, arrays provide a convenient way to store and manage multiple elements efficiently.

Anatomy of an Array

The basic idea of an array is to group multiple items under a single variable. Each item stored in an array is called an array element and is associated with an index that starts from 0. The first element is stored at index 0, the second at index 1, and so forth.

Example:

$fruits = array("Apple", "Banana", "Cherry");
echo $fruits[0]; // Output: Apple

Types of Arrays

In PHP, there are three main types of arrays:

  • 1. Indexed Arrays: These are arrays with numeric indexes. The index is automatically assigned, starting from 0. You can also manually assign indexes.
    • $colors = array("Red", "Green", "Blue");
echo $colors[1]; // Output: Green

// Manually assigning indexes
$manualIndex = array(1 => "One", 2 => "Two", 3 => "Three");
echo $manualIndex[2]; // Output: Two

  • 2. Associative Arrays: These arrays use named keys that you assign to them instead of numeric indexes. This allows you to access values using descriptive keys.
    • $ages = array("Alice" => 25, "Bob" => 30, "Charlie" => 35);
echo $ages["Charlie"]; // Output: 35

  • 3. Multidimensional Arrays: These are arrays containing one or more arrays. They allow you to store data in a matrix format, making it easier to manage related data.
    • $students = array(
    array("Alice", 25),
    array("Bob", 30),
    array("Charlie", 35)
);
echo $students[2][0]; // Output: Charlie

Common Array Functions

PHP provides a variety of built-in functions to work with arrays. Here are some commonly used functions:

  • count(): Counts the number of elements in an array.
    • $numbers = array(1, 2, 3, 4, 5);
echo count($numbers); // Output: 5

  • array_push(): Adds one or more elements to the end of an array.
    • array_push($fruits, "Mango");
print_r($fruits); // Output: Array ( [0] => Apple [1] => Banana [2] => Cherry [3] => Mango )

  • array_pop(): Removes the last element from an array and returns it.
    • $lastFruit = array_pop($fruits);
echo $lastFruit; // Output: Mango
print_r($fruits); // Output: Array ( [0] => Apple [1] => Banana [2] => Cherry )

  • array_keys(): Returns all the keys of an associative array.
    • $keys = array_keys($ages);
print_r($keys); // Output: Array ( [0] => Alice [1] => Bob [2] => Charlie )

  • in_array(): Checks if a value exists in an array.
    • if (in_array("Alice", $keys)) {
    echo "Alice is in the array.";
} else {
    echo "Alice is not in the array.";
}
// Output: Alice is in the array.

Iterating Through Arrays

You can iterate through arrays using loops. The foreach loop is particularly useful for working with arrays:

$fruits = array("Apple", "Banana", "Cherry");
foreach ($fruits as $fruit) {
    echo $fruit . " ";
}
// Output: Apple Banana Cherry

You can also iterate through associative arrays using keys:

foreach ($ages as $name => $age) {
    echo "$name is $age years old.
";
}
// Output: Alice is 25 years old.
//         Bob is 30 years old.
//         Charlie is 35 years old.
