PHP BASICS

Introduction

PHP is the web development language written by and for web developers. PHP stands for PHP Hypertext Pre-preprocessor. Originally named Personal Home Page, PHP is a server-side scripting language, typically used to create web applications in combination with a web server, such as Apache.

Tokens

A token is the basic component of source code in a programming language. It is the smallest different, individual, and independent unit used in any programming language. Tokens play an important role in the development of a program.
The different types of tokens used in PHP are:
1. Keywords
2. Identifiers
3. Constants
4. Operators
5. Strings

Keywords

Below is a table listing some commonly used PHP keywords and their descriptions:

Keyword Description

if Conditional statement to execute code if a condition is true.
else Executes code if the preceding if condition is false.
while Loops through code as long as a specified condition is true.
for Loops through code a specific number of times.

function Defines a reusable function.
return Returns a value from a function.
class Defines a class, which is a blueprint for objects.
public Defines a public access modifier for class members.
private Defines a private access modifier for class members.
protected Defines a protected access modifier for class members.
new Creates a new instance of a class.
try Begins a block of code to test for exceptions.
catch Catches exceptions thrown in a try block.
namespace Defines a namespace for organizing code and avoiding name conflicts.

Identifiers

An identifier is simply a name in PHP. Identifiers are used to define variables, functions, constants, and classes with user-defined names.
The rules for naming identifiers in PHP are:
1. Only alphabet characters, digits, and underscores are permitted.
2. The first character must be an alphabet or underscore. For example, $abc, $_abc are valid identifier names, but $4abc, S'abo etc. are not valid identifier names.
3. The name cannot have any special character besides underscore. For example, $a_b is valid, but $a'b is not a valid identifier name.
4. Identifiers must not contain white space. If you need space in identifier name, you must use an underscore. For example, $basic_salary is a valid identifier name.
5. The first character of an identifier must be either an ASCII letter (uppercase or lowercase), the underscore character, or any of the characters between ASCII 0x7F and ASCII 0xFF. After the initial character, these characters and the digits 0-9 are valid.

Variables

A variable is a user-defined name given to a storage area that may be used to store a data value. A variable carries a value with it. Variables are quantities whose value can vary during the execution of the program; it is a way to access the computer's memory. Variables act as containers that store information for later use in your scripts.
The various rules to define the variables in PHP are:
1. All variables in PHP are denoted with a leading dollar sign ($). For example, $name is a valid variable name, but name is not a valid variable name in PHP.
2. Variables in PHP are case-sensitive, which means $name and $Name are treated as different variables.
3. It can contain only alphanumeric characters and underscores (A-Z, a-z, 0-9, and underscore).
4. The first letter after $ of the variable name must be a letter or an underscore. For example, $name, $_name are valid variable names, but $4name is not a valid variable name.
5. Variable names must not be keywords. Although it is not officially illegal to use keywords as variable names, it's conventionally considered bad practice. For example, $if, $break, $final etc. contain keywords and are valid variable names, but it is not good practice for programmers to take them as variable names.
6. Variable names must not contain white spaces. If you need space in variable names, you must use an underscore. For example, $basic_salary is a valid identifier name but $basic salary is not.

Constants

Constants are like variables but once defined, their values cannot be changed during the programâ€™s execution. They are defined using the define() function or the const keyword.
The rules for defining constants include:
1. Constant names do not start with a dollar sign ($).
2. Constants are case-sensitive by default, though they can be made case-insensitive.
3. Constants cannot be redefined or undefined once set.

Data Types

PHP supports several data types to hold different kinds of information:
1. String: A sequence of characters enclosed in single or double quotes, e.g., "Hello, World!".
2. Integer: A non-decimal number, e.g., 123.
3. Float: A number with a decimal point, e.g., 3.14.
4. Boolean: Represents true or false.
5. Array: A collection of values, e.g., array("apple", "banana", "cherry").
6. Object: An instance of a class that may contain both data and functions.
7. NULL: A variable with no value assigned.
