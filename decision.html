Decision Making

Introduction

Decision making refers to the order of execution of statements based on certain conditions. It tells the computer to execute a particular part of the program only if the condition is true. It is common in many situations where we may have to change the order of execution based on certain conditions. The control structure determines the order in which statements are executed. It is a block of programming that analyzes variables and chooses a direction in which to go based on given parameters.
The branching statements are used to make decisions based on different conditions. In PHP, we have the following five branching statements:
1. if statement
2. if-else statement
3. if-elseif-else statement
4. nested if statement
5. switch statement

1. if Statement

The if statement executes a block of code if the specified condition is true. If the condition is false, the code block is skipped. For example:

                if ($a > $b) {
                    echo "$a is greater than $b";
                }

2. if-else Statement

The if-else statement allows for two blocks of code to be executed: one if the condition is true and another if the condition is false. For example:

                    if ($a > $b) {
                        echo "$a is greater than $b";
                    } else {
                        echo "$a is not greater than $b";
                    }

3. if-elseif-else Statement

The if-elseif-else statement is used when there are multiple conditions to evaluate. You can chain multiple elseif statements between the if and else. For example:

                if ($a > $b) {
                    echo "$a is greater than $b";
                } elseif ($a < $b) {
                    echo "$a is less than $b";
                } else {
                    echo "$a is equal to $b";
                }

4. Nested if Statement

A nested if statement is an if statement inside another if statement. This allows for more complex decision-making structures. For example:

                if ($a > $b) {
                    echo "$a is greater than $b";
                } elseif ($a < $b) {
                    echo "$a is less than $b";
                } else {
                    echo "$a is equal to $b";
                }

5. switch Statement

The switch statement is an alternative to using multiple if statements. It allows you to evaluate a single expression against multiple possible values. For example:

                    switch ($fruit) {
                        case "apple":
                            echo "You chose an apple";
                            break;
                        case "banana":
                            echo "You chose a banana";
                            break;
                        default:
                            echo "Unknown fruit";
                    }
